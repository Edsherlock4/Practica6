FRONTEND

El frontend se refiere a la parte de una aplicación o sitio web con la que los usuarios interactúan directamente. Incluye el diseño, la estructura y la funcionalidad visual de la interfaz, asegurando que la experiencia del usuario (UX) sea intuitiva y eficiente.

Fundamentos del Frontend

Para desarrollar un frontend efectivo, se utilizan los siguientes principios y tecnologías clave:

  1. HTML (HyperText Markup Language): Estructura la página web.
  2. CSS (Cascading Style Sheets): Da estilo y diseño a los elementos.
  3. JavaScript: Añade interactividad y dinamismo a la interfaz.
